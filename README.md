# My Portfolio

This is my personal portfolio project built with **HTML, CSS, Flask, and SQLite**.

**Author:** Shaik Fariya

## Run Locally

1. Install Python 3.11+
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Start the app:
   ```bash
   python server.py
   ```
4. Open in browser:
   ```
   http://localhost:8080
   ```

## Free HTTPS Deployment

This project is configured for **Render** free deployment with automatic **HTTPS**.

1. Push the project to GitHub
2. Create a new **Web Service** on Render
3. Select this repository
4. Choose **Docker** environment
5. Deploy

Render will automatically provide an `https://` URL for the live site.
